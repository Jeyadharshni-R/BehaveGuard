# 🔐 BehaveGuard

BehaveGuard is a lightweight security system that detects unauthorized access using file monitoring, anomaly detection, and face capture.

## 🚀 Features
- Real-time file monitoring
- Intruder detection
- Webcam image capture
- Email & Telegram alerts
- Remote approval system
- Auto system lock

## 🛠 Technologies
- Python
- OpenCV
- Watchdog
- Telegram Bot API
